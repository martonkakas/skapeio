# Asset credits

## Discobolus

- **Work:** Discobolus (the Discus Thrower), scan of cast KAS1549
- **Digitization:** SMK – National Gallery of Denmark
- **Rights:** [Public Domain Mark 1.0](https://creativecommons.org/publicdomain/mark/1.0/)
- **Source:** [SMK Open artwork record](https://open.smk.dk/en/artwork/image/KAS1549)
- **Local file:** `public/models/discobolus-mesh.glb`
- **Processing:** the museum STL was reduced to 36,000 triangles and converted to GLB for real-time wireframe rendering.

## Earth imagery

The Earth component uses a monochrome land mask derived from NASA Earth Observatory / NASA Goddard Space Flight Center's [Blue Marble: Next Generation July base map](https://science.nasa.gov/earth/earth-observatory/blue-marble-next-generation/base-map/). The source image is used only as geometry data for the continent mesh and coastline shader; it is not displayed photographically.
